# 如何使用 git 版本控制

### git clone - 複製專案
```
git clone <repository_url>
```

### git config - 設定姓名email
```
git config user.name "<name>"
git config user.email "<email>"
```

### git add
```
git add .
```

### git diff
```
git diff --staged
```

### git commit
```
git commit -m "<commit_message>"
```

### git push
```
git push origin master
```
